# Semem Build Scripts

This directory contains build and deployment scripts for the Semem project.

## build-docs-site.js

Builds the complete documentation site structure for GitHub Pages deployment.

### What it does:

1. **Creates site structure** in `_site/` directory
2. **Generates landing page** with navigation between manual and API docs
3. **Creates HTML wrappers** for all manual markdown files
4. **Sets up proper navigation** between sections
5. **Creates .nojekyll** to disable Jekyll processing

### Site Structure:

```
_site/
├── index.html              # Landing page with navigation
├── .nojekyll              # Disables Jekyll processing
├── manual/                # User manual
│   ├── index.html        # Manual index wrapper
│   ├── config.html       # Configuration guide wrapper
│   ├── mcp-tutorial.html # MCP tutorial wrapper
│   ├── *.md             # Original markdown files
│   └── *.html           # HTML wrappers for each .md file
└── api/                 # JSDoc API documentation
    └── ... (generated by JSDoc)
```

### Usage:

```bash
# Build complete documentation site
npm run docs:build

# Deploy to GitHub Pages
npm run ghp
```

### Features:

- **Responsive design** with GitHub-style CSS
- **Automatic link conversion** from .md to .html
- **Error handling** for missing content
- **Professional styling** with navigation bars
- **Mobile-friendly** responsive layout
